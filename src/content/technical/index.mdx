# Technical Documentation

Developer-facing documentation for the Miwim decentralized proxy infrastructure.

> This documentation describes the **planned architecture and design**. Components are under active development. Implementation details may change.

## Planned Tech Stack

| Component | Technology | Purpose |
|-----------|-----------|---------|
| Provider Agent | Go + Singbox core | Runs on provider machines, manages proxy service |
| Backend API | Go (Fiber) | Node registry, routing, subscriptions |
| Database | PostgreSQL | Users, nodes, subscriptions, logs |
| Cache | Redis | Health scores, session management, rate limiting |
| Monitoring | Prometheus + Grafana | Node health, network metrics |
| Config Generator | Go | Dynamic Singbox config per user |
| Smart Router | Go | Optimal node selection per request |

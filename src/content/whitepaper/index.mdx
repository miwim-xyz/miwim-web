# Miwim Whitepaper

> Decentralized Proxy Infrastructure — Powered by DePIN

## Abstract

Miwim is a decentralized proxy network that combines DePIN economics with consumer-grade VPN and B2B residential IP services. By incentivizing independent providers to contribute bandwidth through the $MIWIM token, Miwim creates a trustless, censorship-resistant proxy infrastructure owned by its participants.

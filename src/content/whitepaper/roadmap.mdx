# Roadmap

## Q3 2026 — Foundation

**Goal**: Validate demand, build MVP, bring real traction data to investor conversations.

- MVP launch with 10+ decentralized nodes across 3-5 countries
- 100 beta users with real usage data
- Provider agent v1 with health scoring and auto-failover
- Android app connected to decentralized backend
- Seed round ($100K-$500K)
- Landing page and dual waitlist (users + genesis nodes)

## Q4 2026 — Growth

**Goal**: Expand platform reach and begin B2B revenue generation.

- iOS and desktop applications
- White-label API for proxy resellers
- 50+ active nodes globally
- 1,000+ users
- First B2B clients generating API revenue
- Enhanced smart routing with geographic optimization
- Provider dashboard with earnings tracking

## Q1 2027 — Token Launch

**Goal**: Launch $MIWIM token and transition to decentralized economics.

- Token contract deployment (testnet → mainnet)
- $MIWIM Token Generation Event (TGE)
- Provider staking goes live
- On-chain bandwidth proof mechanism
- Node reward emission begins
- DEX liquidity provision
- Community airdrop to beta testers

## Q2 2027 — Scale

**Goal**: Achieve network effects and establish market position.

- 500+ global nodes across 50+ countries
- 10,000+ active users
- 50+ B2B white-label clients
- Decentralized marketplace for bandwidth trading
- DAO governance activation
- Residential IP coverage in major markets
- Enterprise-grade SLA tiers

## Beyond Q2 2027

- Multi-chain token deployment
- Mobile provider agent (earn while idle)
- Advanced privacy features (SGX/SEV secure enclaves)
- Open-source protocol specification
- Community-driven geographic expansion

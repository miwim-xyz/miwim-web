## Token Utility — 5 Demand Sinks

### 1. Pay — User Discount (20% off)
Users who pay with $MIWIM receive a 20% discount on subscriptions. A $10/month plan costs only $8 equivalent in tokens. This creates persistent buy pressure from consumer demand.

### 2. Stake — Provider Collateral
Every node provider must stake a minimum of 5,000 $MIWIM to participate. Staked tokens are locked while the node is active. At 500 nodes: 2.5M tokens locked in provider stakes.

**Slashing**: 10% of stake is burned if SLA is violated (>5% downtime in a month).

### 3. Earn — Bandwidth Rewards
Providers earn $MIWIM proportional to bandwidth served and quality score. Earnings are denominated in USD terms (stable regardless of token price) and paid out in tokens using 24-hour TWAP pricing.

### 4. Govern — Network Parameters
Token holders vote on:
- Node reward distribution ratios
- Geographic incentive multipliers
- Fee structure changes
- Ecosystem fund grant allocation

### 5. Burn — Revenue Buyback
10% of all platform revenue is used for monthly token buyback via on-chain TWAP orders. Bought-back tokens are partially burned (5%) and partially redistributed to top-performing nodes (5%).

## Price Stability Mechanisms

| Mechanism | How It Works |
|-----------|-------------|
| Emission-to-revenue cap | Fewer tokens minted per GB if price drops below threshold |
| Revenue buyback | 10% of revenue buys tokens monthly via TWAP |
| Provider reward vesting | 30-day vest on all reward tokens |
| Low initial float | Only ~6% circulating at TGE |
| Staking lock | Provider + B2B stakes locked while active |

## Hybrid Oracle Pricing

All services are priced in USD. Token is an alternative payment method:

```
Plan price:      $10/month (stable, set in USD)
Fiat payment:    $10 via Stripe/stablecoin
Token payment:   $10 × 0.80 = $8 equivalent in $MIWIM
Token amount:    $8 / TWAP_24h_price
```

Using 24-hour TWAP instead of spot price prevents flash-crash exploits.

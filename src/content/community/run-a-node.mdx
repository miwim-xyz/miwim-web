## Planned Requirements

### Datacenter Nodes (Backbone)
| Requirement | Minimum | Recommended |
|-------------|---------|-------------|
| OS | Linux (Ubuntu 20.04+) | Ubuntu 22.04 LTS |
| CPU | 1 vCPU | 2 vCPU |
| RAM | 512 MB | 1 GB |
| Bandwidth | 100 Mbps | 500+ Mbps |
| Monthly traffic | 500 GB | Unlimited |

### Residential Nodes (Edge)
| Requirement | Minimum | Recommended |
|-------------|---------|-------------|
| Connection | 50 Mbps home broadband | 200+ Mbps |
| Device | Any always-on Linux device | Raspberry Pi 4+ or mini PC |
| Network | Outbound port 443 open | No special config needed |

> Staking requirement (5,000 $MIWIM per node) will begin at mainnet token launch, targeted for Q1 2027. Beta providers will not need to stake.

## FAQ

**When can I start running a node?**
The provider agent is under development. Genesis node operators will be the first to receive access during beta (targeted Q3 2026).

**Do I need to stake tokens during beta?**
No. Staking begins at mainnet token launch (Q1 2027). Beta providers earn future airdrop allocation instead.

**Can I run multiple nodes?**
Yes. Each node will need its own credentials and must run on a separate IP address.

**Is my home IP exposed to users?**
No. Residential nodes will use reverse tunnels — users connect to a relay, which forwards traffic through the tunnel. Your home IP is never in user configurations.

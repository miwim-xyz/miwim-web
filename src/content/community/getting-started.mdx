# Getting Started

Miwim is currently in **pre-launch**. The network is not yet live for public use.

## How to Get Involved Now

### 1. Join the Waitlist
Visit [miwim.io](/) and sign up. Waitlist members will receive:
- Priority access when the network launches
- Airdrop eligibility for $MIWIM tokens
- Early beta invitations

### 2. Follow Development
Stay updated on progress through our community channels (coming soon).

### 3. Apply as a Genesis Node Provider
If you have idle server capacity or home broadband, [apply as a genesis node operator](/#waitlist). Genesis providers will receive founding rewards and bonus token allocation once the network launches.

# Community

Welcome to the Miwim community. The project is in pre-launch — here's how to get involved early.

## Get Involved

### As a User
1. [Join the waitlist](/#waitlist) for early access and airdrop eligibility
2. Follow development updates (social channels coming soon)

### As a Node Provider
1. [Apply as a genesis node operator](/#waitlist)
2. Review the [planned Provider Agent spec](/docs/technical/provider-agent)
3. Prepare your server (minimum: 1 vCPU, 512MB RAM, 50 Mbps)

### As a Developer
1. Read the [Technical Documentation](/docs/technical) to understand the architecture
2. Check the [Whitepaper](/docs/whitepaper) for the full vision

# Community

Welcome to the Miwim community. There are several ways to get involved.

## Links

- **Twitter / X**: [@miwim_network](#) — Project updates, DePIN insights, and announcements
- **Telegram**: [Miwim Community](#) — Discussion, support, and feedback
- **Discord**: [Miwim Discord](#) — Developer chat and community events
- **GitHub**: [miwim-network](#) — Open-source components and contributions

## Getting Involved

### As a User
1. [Join the waitlist](/#waitlist) for early access
2. Follow us on Twitter for launch announcements
3. Join the Telegram group for community discussion

### As a Provider
1. [Apply as a genesis node operator](/#waitlist)
2. Review the [Provider Agent documentation](/docs/technical/provider-agent)
3. Prepare your server (minimum: 1 vCPU, 512MB RAM, 50 Mbps)

### As a Developer
1. Star our GitHub repositories
2. Read the [Technical Documentation](/docs/technical)
3. Join the Discord developer channel
